analytics Ferramentas de Engenharia de Dados Selecione uma ferramenta para simular e otimizar custos.

Nosso objetivo é ajudá-lo a encontrar a melhor configuração com o menor custo possível para suas necessidades de engenharia de dados.

Visão Geral

Este dashboard é uma ferramenta para simular e otimizar custos de serviços AWS focados em cargas de trabalho de engenharia de dados. Comece clicando em "Calcular Custo EC2" para simular o preço de uma instância de servidor.
